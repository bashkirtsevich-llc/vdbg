Mozaa PC Emulator

(Эмулятор ПЭВМ с ОС ДОС Моза)

Основан на коде bochs

Создадим хороший эмулятор старого ПК на Delphi!

На данном этапе поддерживает практически полную эмуляцию 486DX + набор команд P5, но
с FPU не все гладко. На данном этапе именно это - основная проблема.

Немного подчищен код, исправлены основные замеченные ошибки
